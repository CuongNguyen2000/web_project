# web_project

- To run this prject the system need to install
  - NodeJS
  - Yarn (Recommended):
    To install yarn using command: npm install yarn --global
- Install dependency before run this project:<br>
  yarn install<br>
  or<br>
  npm install<br>
- To run this project:<br>
  yarn start<br>
  or<br>
  npm start<br>
